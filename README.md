## Eternal Check

* Eternal Check verifies if an ip is vulnerable to the smb vulnerabilities 
- Eternal Blue 
- Eternal Romance 
- Eternal champion 
- Eternal synergy
- Erraticgopher SMB v1 Exploit (Windows XP & Server 2003)
